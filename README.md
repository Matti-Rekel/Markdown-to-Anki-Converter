# Markdown-to-Anki-Converter
Converts my Anki-Notes in Markdown to a text file which can be importet to Anki

--- 
## Examples:
For Examples see Example Folder

---
## Install:
It should work on Windows and Linux you just have to compile the Converter.java file and Execute it with the Path of the Markdown-File you want converted. On the first Execution it will prompt you to enter the directory of Ankis-Media folder which is needed to support image export. You can find this in Anki under "tool/check Media/View Files".

After following these Steps the file can be imported from Anki. Just press import File and then select your File from the Output-Folder.
