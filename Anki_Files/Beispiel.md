### This is an example card with all currently supported features:

Hinweis: This shows up as an optional hint on the front of the Card

This is the Answer

Every field can stretch over multiple lines but only Informationen and answer currently also allow empty lines between text.

There are multiple highlights supported *Highlight1*, **Highlight2** and ***Highlight3***

Also features are ordered as well as unordered lists
- line one
    - indented line
- line two

1. first line
2. second line
    1. first indented line

also supported are math:
for small equations: $ a² + b² = c² $

And big equations which become scrollable in Anki on mobile:
$$
R = \frac{1}{2} \int_0^1 x (1-x) f''(x) dx = \frac{1}{12} f''(\xi )
$$

And there is basic support for code:
currently either with Java or without additional info in Markdown

````java
System.out.println(x);
````


Informationen: Optional side on the back of the Card

Quelle: Additional optional side on the back of the Card
